Section 02

Templates de relance CDS

3 niveaux de relance, chacun avec une approche differente. Messages personnalises dynamiquement avec les donnees du prospect.

J+3 — Valeur

Partager un insight ou article pertinent

[Prenom], en passant — j'ai ecrit un guide sur [sujet lie a leur metier]. Ca pourrait vous faire gagner du temps sur [douleur identifiee].

[lien article blog]

— Aissa
J+7 — Preuve sociale

Cas client similaire a leur situation

[Prenom], rapide retour : j'ai fini un projet pour [type entreprise similaire] a [ville proche]. [Resultat concret en 1 phrase].

Si c'est un sujet pour [entreprise], je suis dispo cette semaine pour en discuter.

— Aissa
J+14 — Porte de sortie

Message court, derniere relance respectueuse

[Prenom], je ne veux pas insister — si le timing n'est pas bon, aucun souci.

Si jamais le sujet revient, le diagnostic gratuit est toujours la : aissabelkoussa.fr/diagnostic

— Aissa
Section 03

Script CLI

Interface en ligne de commande pour gerer le funnel de prospection au quotidien.

# Relances du jour — le matin
python follow-up.py

# Ajouter un prospect
python follow-up.py --add

# Mettre a jour un statut
python follow-up.py --status 3 rdv_pris

# Voir tous les prospects
python follow-up.py --list

# Stats funnel
python follow-up.py --stats

# Importer les 30 prospects BTP
python follow-up.py --import-html
