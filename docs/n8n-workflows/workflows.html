Workflows a creer

1. lead-intake.json — Reception et scoring

  • Trigger : Webhook (POST depuis l'API contact Next.js)
  • Actions : Parser le payload, creer entree Notion/Airtable, notification WhatsApp si CHAUD, email nurturing si TIEDE/FROID

2. sequence-nurturing.json — Sequence email automatique

  • Trigger : Cron quotidien
  • J+1 : Email "Votre diagnostic"
  • J+3 : Email "3 questions pour cadrer votre projet"
  • J+7 : Email "Etude de cas : plombier +10h/semaine"
  • J+14 : Email "Derniere chance — creneau disponible"

3. maintenance-transition.json — Transition mois 3

  • Trigger : Cron (clients dont la periode se termine dans 15 jours)
  • Envoyer template avec metriques, planifier appel, relance WhatsApp si pas de reponse

4. monthly-report.json — Rapport mensuel

  • Trigger : Cron (1er de chaque mois)
  • Fetcher metriques Vercel/UptimeRobot, generer rapport, envoyer aux clients actifs

5. whatsapp-notif.json — Notification lead chaud

  • Trigger : Appele par lead-intake quand score >= 55
  • Message WhatsApp + rappel calendrier

Stack technique recommandee

  • n8n : self-hosted ou cloud
  • Resend : emails transactionnels (deja configure)
  • Notion/Airtable : CRM pipeline
  • UptimeRobot : monitoring uptime (gratuit)
  • WhatsApp Business API : ou Twilio
