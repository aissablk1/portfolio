02 — SEO classique vs GEO : ce qui change

Le GEO (Generative Engine Optimization) est la pratique d'optimiser le contenu pour être cité, référencé ou recommandé par les moteurs IA : Google AI Overview, ChatGPT, Perplexity, Gemini.

CritèreSEO classiqueGEO (nouveau)

Métrique cléPosition, CTRTaux de citation, taux de mention
Levier principalBacklinks, mots-clésDensité factuelle, données structurées
FraîcheurImportance modéréeCritique (cycle 7-14 jours)
FormatLibreAnswer-first, listes, tableaux
AutoritéDomain AuthorityE-E-A-T + preuves sociales
IndexationGoogle uniquementGoogle + Bing + LLM (multi-moteur)

Être cité dans un AI Overview = +35% de clics organiques

L'objectif n'est plus seulement de ranker. C'est d'être la source que l'IA choisit de citer. 97% des sources citées dans les AI Overviews proviennent des 20 premiers résultats organiques — sans bon SEO de base, zéro chance d'être cité par l'IA.

80% du GEO repose sur un bon SEO. Les deux sont complémentaires, pas opposés.

03 — Bonne nouvelle : le SEO local est protégé

Seulement 7,9% des recherches locales déclenchent un AI Overview

Le Pack Local (Google Maps) reste le format dominant pour "consultant IA Albi" ou "site web artisan Tarn". Raisons : les requêtes locales nécessitent des données temps réel (horaires, proximité, disponibilité) que l'IA ne peut pas synthétiser.

Mais l'IA arrive dans le local aussi

05 — Meilleures pratiques GEO 2026

Structure du contenu

  • Answer-first : commencer chaque section par la réponse directe
  • 44,2% des citations LLM viennent des premiers 30% du contenu
  • Listes, tableaux, étapes numérotées (pas de blocs compacts)
  • Statistiques sourcées (+40% de visibilité IA)
  • FAQ alignées sur les vrais prompts utilisateurs

Données structurées (Schema.org)

  • JSON-LD obligatoire : Article, FAQPage, HowTo, LocalBusiness, Person
  • 65% des pages citées dans les AI Overviews ont du balisage structuré
  • Schema stacking : combiner plusieurs types sur une même page
  • Nouveau : llms.txt à la racine (guide les LLM vers les pages clés)

Fraîcheur du contenu

  • Cycle de mise à jour : 7 à 14 jours
  • Au-delà, le contenu perd sa priorité de citation
  • 1-2 nouveaux contenus par semaine minimum
  • Mettre à jour les articles existants (pas seulement créer)

Multi-moteurs IA

  • Google AI Overview : SEO classique + schema.org
  • ChatGPT : indexation Bing + contenu encyclopédique
  • Perplexity : fraîcheur + sources communautaires + llms.txt
  • Gemini : écosystème Google (GBP, YouTube)
